# Roadmap  E-Shop Core API

Цей план відображає пріоритетні функції для реалізації протягом найближчих 3–4 тижнів.

| Період | Тиждень | Завдання |
| :--- | :--- | :--- |
| **Найближчі 7 днів** | Тиждень 3 | Додати динамічний **GET-ендпоінт** для одного товару: `GET /items/<id>`. Додати HTML-сторінку для відображення детальної інформації про товар. |
| **Наступні 7 днів** | Тиждень 4 | Базова **авторизація**. Створити таблицю `users` у БД. Реалізувати ендпоінти `POST /login` та `POST /register` для Адміністратора. |
| **Наступні 7 днів** | Тиждень 5 | Реалізація **CRUD** для Адміністратора. Створення ендпоінтів `POST /items`, `PUT /items/<id>`, `DELETE /items/<id>`. |
| **Перспектива** | Тиждень 6 | Додати функціональність "Кошик" (Cart) та ендпоінти для оформлення замовлення. |
