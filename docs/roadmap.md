# Roadmap  E-Shop Core API

Цей план відображає пріоритетні функції для реалізації протягом найближчих 3–4 тижнів.

## Тиждень 2
- Реалізувати CRUD для товарів (додавання, редагування, видалення).  
- Створити окремий модуль `products` із власними маршрутами.

## Тиждень 3
- Додати оформлення замовлення (`orders`).  
- Після створення замовлення викликати сповіщення (`notifications`).

## Тиждень 4
- Реалізувати авторизацію користувачів (`auth`): реєстрація, логін.  
- Зв’язати замовлення з користувачем, щоб кожен бачив лише свої.

## Тиждень 5
- Створити просту адмін-панель для керування товарами та замовленнями.  
- Додати обмеження доступу лише для ролі **admin**.
